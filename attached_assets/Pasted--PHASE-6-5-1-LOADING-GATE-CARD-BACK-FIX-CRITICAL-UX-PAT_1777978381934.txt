# =========================

# SECTION 1 — FIX “EMPTY REVEAL” BUG

# =========================

## PROBLEM:

User swipes up before API finishes → reveal runs with no cards → UI flashes back to main screen

---

## SOLUTION: ADD LOADING GATE

We block reveal until cards are ready.

---

## 1.1 — ADD LOADING STATE

In your pack flow controller:

```js
let isPackReady = false;
let pendingCards = null;
```

---

## 1.2 — MODIFY BUY PACK FLOW

```js
async function handleBuyPack(setId) {
  isPackReady = false;

  openPackInteraction(setId); // animation starts immediately

  const cards = await fetchSetCards(setId);

  pendingCards = cards;
  isPackReady = true;

  hidePackLoadingIndicator();
}
```

---

## 1.3 — INTERCEPT SWIPE UP

Wherever swipe-up triggers reveal:

```js
function onSwipeUp() {
  if (!isPackReady) {
    showPackLoadingIndicator();
    return; // BLOCK reveal
  }

  startReveal(pendingCards);
}
```

# SECTION 2 — PACK LOADING INDICATOR (IN-OVERLAY)

# =========================

## 2.1 — ADD UI ELEMENT

Inside pack overlay:

```html
<div id="pack-loading-indicator" class="hidden">
  <div class="loading-dots">
    <span></span><span></span><span></span>
  </div>
  <p>Loading cards...</p>
</div>
```

---

## 2.2 — CSS (SMOOTH DOT ANIMATION)

```css
#pack-loading-indicator {
  position: absolute;
  bottom: 80px;
  width: 100%;
  text-align: center;
  color: rgba(255,255,255,0.8);
  z-index: 20;
}

.loading-dots span {
  display: inline-block;
  width: 6px;
  height: 6px;
  margin: 0 3px;
  background: white;
  border-radius: 50%;
  animation: dotPulse 1.2s infinite;
}

.loading-dots span:nth-child(2) { animation-delay: 0.2s; }
.loading-dots span:nth-child(3) { animation-delay: 0.4s; }

@keyframes dotPulse {
  0%, 80%, 100% { opacity: 0.3; transform: scale(1); }
  40% { opacity: 1; transform: scale(1.4); }
}
```

---

## 2.3 — JS CONTROL

```js
function showPackLoadingIndicator() {
  document.getElementById('pack-loading-indicator').classList.remove('hidden');
}

function hidePackLoadingIndicator() {
  document.getElementById('pack-loading-indicator').classList.add('hidden');
}
```

---

## RESULT:

✔ If user swipes too early → loading dots appear
✔ No UI flicker
✔ No confusion
✔ Reveal ONLY happens when ready

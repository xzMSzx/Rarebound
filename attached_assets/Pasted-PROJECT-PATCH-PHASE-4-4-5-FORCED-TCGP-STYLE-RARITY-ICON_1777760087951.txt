PROJECT PATCH — PHASE 4.4.5  
FORCED TCGP-STYLE RARITY ICON SYSTEM (NO TEXT SYMBOLS)

This patch replaces the entire rarity symbol system with filled SVG icons identical in concept to Pokémon TCG Pocket.

The previous Unicode symbol system must be removed completely.

The assistant must NOT replace SVG icons with Unicode characters or CSS-colored text.

SVG icons are mandatory.

---

CRITICAL RULES

The following implementations are explicitly forbidden:

Unicode rarity symbols (◇ ★ ✦ etc)  
CSS-only symbol rendering  
text-based rarity indicators  

The system must use inline SVG icons only.

---

SECTION 1 — REMOVE EXISTING SYMBOL SYSTEM

Delete the following:

ui/raritySymbols.js  
all Unicode symbols from cardRenderer.js  
all Unicode symbols from fullscreenOverlay.js  
all rarity-symbol text CSS

Remove any code rendering:

◇  
★  
✦  

The rarity system must be rebuilt using SVG icons.

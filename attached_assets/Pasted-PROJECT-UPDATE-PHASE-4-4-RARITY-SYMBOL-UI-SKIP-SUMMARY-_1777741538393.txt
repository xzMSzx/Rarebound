PROJECT UPDATE — PHASE 4.4 RARITY SYMBOL UI + SKIP SUMMARY LAYOUT

This update introduces the Pokémon-style rarity symbol system and improves the Skip Pack summary layout for 10-card packs.

The internal rarity backbone implemented in Phase 4.3 must remain unchanged.

This update affects only UI display.

---

CRITICAL RULES

Do NOT modify:

engine RNG system  
pack generation logic  
pity system  
card pool manager  
rarity mapper  
set probability tables  
overlay animation system  
swipe controller  

Only modify UI display logic.

---

SECTION 1 — DEFINE RARITY SYMBOL SYSTEM

Replace the temporary rarity labels (C, U, R, H, DR, IR, UR, SR, HR) with a symbol-based system.

RARITY → SYMBOL

common → ◇  
uncommon → ◇◇  

rare → ★  
holoRare → ★  

doubleRare → ★★  
illustrationRare → ★★  

ultraRare → ★★★  

specialIllustrationRare → ✦★  

hyperRare → ✦✦  

This system avoids excessive star stacking while maintaining clear progression.

---

SECTION 2 — CREATE SYMBOL MAPPING FILE

Create:

ui/raritySymbols.js

Example structure:

export const RARITY_SYMBOLS = {
  common: "◇",
  uncommon: "◇◇",
  rare: "★",
  holoRare: "★",

  illustrationRare: "★★",
  ultraRare: "★★★",
  specialIllustrationRare: "✦★",
  hyperRare: "✦✦"
}

---

SECTION 3 — SYMBOL STYLING

Create CSS class:

rarity-symbol

Style guidelines:

font-size: 1.2rem  
font-weight: 500  
letter-spacing: 2px  
opacity: 0.9  

Symbols should be clean and minimal.

Do NOT apply glow or colored outlines.

Card rarity should be identifiable primarily through the card artwork and later holo effects.

---

SECTION 4 — REMOVE CARD GLOW EFFECTS

Any existing card glow effects should be removed.

Glow must NOT be applied to the card container.

Future holographic effects will provide visual excitement.

Rarity symbols remain subtle indicators.

---

SECTION 5 — PREMIUM SYMBOL GRADIENT

For high-tier rarities apply gradient text styling.

Applies only to:

ultraRare  
specialIllustrationRare  
hyperRare

Example styling concept:

background: linear-gradient(90deg,#fbbf24,#fde047)

Use:

background-clip: text  
color: transparent

This gives the symbol a metallic premium look.

---

SECTION 6 — MICRO ANIMATION FOR PREMIUM TIERS

Premium rarity symbols should have subtle animation.

Applies to:

ultraRare  
specialIllustrationRare  
hyperRare

Example concept:

animation: rarityShimmer 3s linear infinite

The animation should resemble a gentle shimmer across the symbol.

Avoid aggressive glow or pulsing.

---

SECTION 7 — UPDATE CARD REVEAL UI

File: fullscreenOverlay.js

Replace rarity text with symbol display.

Example:

New:

Apply the CSS class:

rarity-symbol

SECTION 9 — FIX SKIP SUMMARY FOR 10 CARDS

The Skip Pack summary was designed for 5 cards and currently causes cards to render off-screen.

Update layout to a 2-row grid.

Structure:

Row 1 → 5 cards  
Row 2 → 5 cards

Example layout:

[C][C][C][R][H]  
[H][U][U][C][C]

CSS concept:

display: grid  
grid-template-columns: repeat(5, 1fr)  
gap: 14px  

This ensures all 10 cards are visible.

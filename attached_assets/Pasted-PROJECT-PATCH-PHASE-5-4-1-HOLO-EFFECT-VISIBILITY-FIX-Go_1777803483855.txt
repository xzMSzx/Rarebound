SECTION 3 — ADD HOLO ONLY FOR RARES

Open:

ui/cardRevealAnimator.js

Locate the part where the card is rendered after reveal.

Add a rarity check:

const HOLO_RARITIES = new Set([
"rare",
"doubleRare",
"illustrationRare",
"ultraRare",
"specialIllustrationRare",
"hyperRare"
]);

if (HOLO_RARITIES.has(card.rarity)) {

const rainbow = document.createElement("div");
rainbow.className = "holo-rainbow";

const reflection = document.createElement("div");
reflection.className = "holo-reflection";

cardFace.appendChild(rainbow);
cardFace.appendChild(reflection);

}

Important:

cardFace must be the element that contains the card image.

Do NOT append to the wrapper.

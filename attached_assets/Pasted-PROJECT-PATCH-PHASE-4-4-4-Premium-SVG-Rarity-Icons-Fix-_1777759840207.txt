PROJECT PATCH — PHASE 4.4.4  
Premium SVG Rarity Icons (Fix Invisible Icon Bug)

This patch replaces the temporary Unicode rarity symbols with proper filled SVG icons while avoiding the color: transparent issue that previously caused premium rarity icons to disappear.

The key change is that SVG icons will not rely on currentColor.  
Instead they will use explicit fills and gradients inside the SVG, which remain visible even inside gradient containers.

This allows the simulator to achieve a Pokémon TCG Pocket–style rarity system.

SECTION 2 — CREATE SVG RARITY ICON SYSTEM

Create new file:

This file defines all rarity icons as inline SVG elements.

Implementation:

SECTION 5 — PREMIUM RARITY GRADIENT

Add gold gradient for high-tier rarities.

.premium-star {

fill: url(#rarityGoldGradient)

}

Define the gradient in index.html inside the root SVG defs section:

<svg width="0" height="0" style="position:absolute">
<defs>

<linearGradient id="rarityGoldGradient" x1="0%" y1="0%" x2="100%" y2="100%">
<stop offset="0%" stop-color="#ffd700"/>
<stop offset="100%" stop-color="#fff3a3"/>
</linearGradient>

</defs>
</svg>

This gives premium icons a gold shine similar to Pokémon TCG Pocket.

PROJECT PATCH — PHASE 5.7.1
REAL BOOSTER PACK ART + AUTHENTIC CARD BACK

Goal:

Upgrade the pack opening visuals to feel like a real Pokémon TCG pack opening.

Current issues to fix:

• Pack shows only the set logo instead of full booster artwork
• Tear seam is not aligned with the foil seal
• Card stack uses grey placeholder cards
• Missing authentic Pokémon card back design

This patch upgrades those elements without touching the simulation engine.

---

CRITICAL RULES

Do NOT modify:

engine RNG
packSimulation.js
pity system
rarity tables
cardPoolManager
rarityMapper
cardRevealAnimator reveal logic
reverse holo system
tilt system
audioManager
rarity glow system
reveal cascade

Only UI assets and pack-opening visuals may be changed.

---

SECTION 1 — CREATE PACK ART DIRECTORY

Create folder:

/assets/packs

This will store real booster pack artwork.

Expected filenames:

sv4pt5.png
swsh7.png
swsh11.png
sv3pt5.png
sv2.png

Each file must correspond exactly to the simulator's setId.

Example:

Paldean Fates → sv4pt5.png
Evolving Skies → swsh7.png

---

SECTION 2 — ADD REAL PACK ART

Place real booster pack artwork in the directory.

Sources for high quality images:

Bulbapedia
Pokémon TCG official site
PokéBeach

The artwork must include:

• Pokémon logo
• set name
• featured Pokémon artwork
• foil wrapper seam at the top

Only one pack artwork per set is needed.

---

SECTION 3 — LOAD PACK ART IN CONTROLLER

Open:

ui/packOpeningController.js

Replace the existing logo URL system.

Before:

const packImage = `https://images.pokemontcg.io/${setId}/logo.png`

After:

const packImage = `/assets/packs/${setId}.png`

Then assign:

packImg.src = packImage

This loads the correct booster pack artwork.

---

SECTION 4 — BOOSTER PACK STYLING

Edit style.css and update the pack container.

.booster-pack {

position: relative;
width: min(65vw, 280px);
aspect-ratio: 2.5 / 3.5;

cursor: grab;

filter: drop-shadow(0 25px 40px rgba(0,0,0,0.55));

transition: transform 0.2s ease;

}

.booster-pack:active {

cursor: grabbing;

}

---

SECTION 5 — PACK IMAGE STYLING

.pack-image {

width: 100%;
height: 100%;

object-fit: contain;

user-select: none;
pointer-events: none;

}

This ensures the artwork fits perfectly inside the wrapper.

---

SECTION 6 — ALIGN THE TEAR SEAM

The tear seam must align with the real foil seal on the pack.

Update seam styling.

.pack-seam {

position: absolute;

top: 4%;
left: 0;

width: 100%;
height: 18%;

cursor: crosshair;

}

This restricts ripping to the foil seam region.

---

Add jagged tear overlay.

.pack-tear {

position: absolute;

left: 0;

height: 40%;

15% 55%, 10% 65%, 5% 55%, 0% 60%

When rip triggers, animate opacity to reveal the tear.

---

SECTION 8 — ADD AUTHENTIC CARD BACK

Create folder:

/assets/cards

Add official Pokémon card back:

pokemon-card-back.png

This is the classic blue Pokémon card back used in the TCG.

---

SECTION 9 — REPLACE PLACEHOLDER CARD STACK

Open:

ui/packOpeningController.js

Locate card stack creation.

Replace grey placeholders with the real card back.

Example:

const cardBack = document.createElement("img")

cardBack.src = "/assets/cards/pokemon-card-back.png"

cardBack.className = "pack-card-back"

stack.appendChild(cardBack)

SECTION 11 — STACKED CARD EFFECT

Add layered cards for realism.

.pack-card-back:nth-child(1) {

transform: translateY(0px) rotate(-2deg);

}

.pack-card-back:nth-child(2) {

transform: translateY(-4px) rotate(1deg);

SECTION 12 — CARD SLIDE UP ANIMATION

Add animation:

@keyframes cardsRise {

}

}

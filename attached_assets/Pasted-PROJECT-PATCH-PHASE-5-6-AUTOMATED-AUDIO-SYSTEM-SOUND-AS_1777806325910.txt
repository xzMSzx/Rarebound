PROJECT PATCH — PHASE 5.6
AUTOMATED AUDIO SYSTEM + SOUND ASSET INSTALLER

Goal:

Introduce a complete **sound effects system** for the simulator while keeping the process **fully automated** so the developer does NOT need to manually search, download, edit, or place audio files.

This patch must:

• Automatically obtain appropriate royalty-free sound effects
• Normalize them into game-ready files
• Install them into the project automatically
• Provide a central audio manager for playback
• Integrate sound triggers into existing gameplay events

All steps must be automated through a setup script.

---

CRITICAL RULES

Do NOT modify:

engine RNG
packSimulation.js
pity system
rarity tables
card pool manager
API loader
card rarity calculations

Only modify or create:

/scripts/setupSounds.js
/assets/sounds/
ui/audioManager.js
style.css (minor optional UI mute toggle styling)
reveal / pack animation trigger points

No gameplay logic should be altered.

SECTION 2 — CREATE AUTOMATED SOUND INSTALLER

Create file:

/scripts/setupSounds.js

Purpose:

Automatically fetch **royalty-free sound effects** from public sound libraries and normalize them for the project.

Sources allowed:

Pixabay Sound Effects
Mixkit
Freesound (CC0 only)

Implementation outline:

1. Define a mapping of required sound effects to search keywords

Example:

```javascript
const SOUND_MAP = {
 packCrinkle: "foil crinkle",
 packTear: "paper tear",
 cardSlide: "paper slide",
 cardFlip: "card flick",
 rareChime: "magic sparkle",
 ultraHit: "magical hit"
}
```

2. For each entry:

• Query Pixabay or Mixkit search endpoint
• Retrieve first CC0 licensed audio preview
• Download file using https request

Example:

```javascript
const https = require("https")
const fs = require("fs")
```

3. Save downloaded files to:

/assets/sounds/

4. Rename automatically using canonical names:

pack-crinkle.mp3
pack-tear.mp3
card-slide.mp3
card-flip.mp3
rare-chime.mp3
ultra-hit.mp3

5. Normalize file size:

If the file exceeds 2 seconds:

Trim to first 1.2 seconds.

If file format ≠ mp3:

Convert to mp3 using ffmpeg if available.

Fallback: keep original format.

---

SECTION 5 — AUDIO PRELOAD

Add function:

```javascript
export function preloadAudio(){

 Object.values(SFX).forEach(sound=>{
  sound.volume = 0.6
  sound.preload = "auto"
  sound.load()
 })

}
```

Safari requires user interaction before sound playback.

Call preloadAudio() on first click or tap.

---

SECTION 6 — SAFE PLAY FUNCTION

Add helper:

```javascript
export function playSound(sound){

 if(!sound) return

 try{
  sound.currentTime = 0
  sound.play()
 }catch(e){
  console.warn("Audio play blocked:", e)
 }

}
```

SECTION 9 — MOBILE SAFETY

Because mobile browsers block autoplay audio:

Add this listener:

```javascript
document.addEventListener("click", ()=>{
 preloadAudio()
}, { once:true })
```

This unlocks audio playback for the entire session.

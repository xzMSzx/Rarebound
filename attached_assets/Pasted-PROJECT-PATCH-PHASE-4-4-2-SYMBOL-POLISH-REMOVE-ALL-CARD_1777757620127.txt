PROJECT PATCH — PHASE 4.4.2 SYMBOL POLISH + REMOVE ALL CARD GLOW

This patch improves the visual quality of rarity symbols, removes all card glow effects, fixes missing symbols in the rarity guide, and ensures consistent styling across the pack reveal UI and collection grid.

This patch modifies UI rendering only.

---

CRITICAL RULES

Do NOT modify:

engine RNG  
pack generation logic  
pity system  
rarity probability tables  
card pool manager  
setProbabilityTables.js  
simulation engine logic  

Only modify UI styling and rendering.

---

SECTION 1 — REMOVE ALL CARD GLOW EFFECTS

Cards should appear as clean objects with no glow.

In style.css locate all glow classes.

Example:

.glow-common  
.glow-rare  
.glow-epic  
.glow-legendary  

Replace their styles with:

box-shadow: none !important  
filter: none !important  

Example final result:

.glow-common,
.glow-rare,
.glow-epic,
.glow-legendary {
box-shadow: none !important;
filter: none !important;
}

This disables glow while preserving animation system compatibility.

SECTION 2 — IMPROVE RARITY SYMBOL APPEARANCE

Current symbols are too small and look like basic ASCII characters.

Upgrade typography.

In style.css add:

.rarity-symbol {

Premium rarities should look visually special without adding glow to cards.

.rarity-ultraRare .rarity-symbol,

.rarity-hyperRare .rarity-symbol {

90deg,

}

Add animation:

0% { background-position: 0% }

100% { background-position: 200% }

}

This creates a subtle premium metallic effect.

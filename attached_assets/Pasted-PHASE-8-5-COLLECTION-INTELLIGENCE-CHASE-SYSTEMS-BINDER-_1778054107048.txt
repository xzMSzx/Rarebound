PHASE 8.5 — COLLECTION INTELLIGENCE, CHASE SYSTEMS, BINDER UX PREMIUMIZATION, WISHLIST EXPERIENCE, EVOLUTION CHAINS, AND CARD NAVIGATION

IMPORTANT:
This patch EXTENDS the existing successful Phase 8 systems.

DO NOT rewrite:

* pack opening
* rarity FX
* holo systems
* audio systems
* preload systems
* reveal systems
* binder persistence
* collection persistence
* statistics logic
* card detail modal architecture

This patch focuses on:

* UX intelligence
* collection psychology
* navigation convenience
* premium collectible feel
* chase-card retention loops

Everything must feel:

* elegant
* premium
* Nintendo/Pokémon-inspired
* psychologically rewarding
* clean on mobile

Avoid:

* clutter
* giant dashboards
* overdesigned realism
* excessive animations

========================================================
SECTION 1 — CLICKABLE STATS SHOWCASE CARDS
==========================================

Current issue:
“Most Valuable Card” and “Rarest Pull” on the Statistics page are dead-end showcases.

This wastes emotional momentum.

========================================================
1.1 — CARD INTERACTION
======================

Make:

* Most Valuable Card
* Rarest Pull

fully clickable.

Behavior:
Tapping the showcase card should:

* open the FULL existing card detail modal
* preserve holo
* preserve tilt
* preserve rarity FX
* preserve value/wishlist functionality

DO NOT create a separate lightweight popup.

Use the SAME card detail modal system used in the binder.

========================================================
1.2 — VIEW IN BINDER BUTTON
===========================

Inside the card detail modal:

Add:
“View In Binder”

small elegant secondary button beneath wishlist button.

Behavior:

* closes modal
* opens relevant set binder
* automatically navigates to correct binder page
* briefly highlights the target card slot

Highlight effect:

* soft gold glow pulse
* 1.2s fade out

========================================================
SECTION 2 — PREMIUM BINDER REALISM
==================================

Current binder:

* clean
* functional
* but visually ambiguous

Goal:
Subtle realism.

NOT:
hyper-realistic 3D binder simulation.

========================================================
2.1 — CARD SLOT IMPROVEMENTS
============================

Improve empty pockets.

Current:

* dark rectangle
* Pokéball icon

New visual design:

* faint sleeve texture
* subtle pocket bevel
* ultra-soft inset shadow
* faint glossy reflection near top-left
* dashed slot outline reduced opacity
* card number more readable

Pokéball icon:

* reduce opacity heavily
* make it feel decorative
* NOT interactive

========================================================
2.2 — PLASTIC SLEEVE EFFECT
===========================

Each slot should have:
VERY subtle semi-transparent sleeve layer.

Effect:

* faint white reflection streak
* slight edge shine
* 4–6% opacity max

Goal:
implied realism.

NOT:
thick fake plastic.

========================================================
2.3 — PAGE MATERIALITY
======================

Binder page background:
Add:

* extremely subtle paper grain
* ultra faint vignette
* slightly warmer dark tones

Should feel:
premium collectible album.

NOT:
flat black div.

========================================================
SECTION 3 — FULL SET CARDDEX SYSTEM
===================================

CRITICAL FEATURE.

Current wishlist system is psychologically broken because users can only wishlist owned cards.

We are fixing this permanently.

========================================================
3.1 — UNOWNED CARD PREVIEW SYSTEM
=================================

Inside every binder:
ALL cards in the set must exist.

Owned cards:

* normal full artwork

Unowned cards:

* hidden preview state

========================================================
3.2 — UNOWNED CARD DESIGN
=========================

DO NOT leave empty black slots.

Instead show:

* faint blurred/silhouetted artwork
  OR
* dark obscured placeholder art

PLUS:

* card number
* rarity indicator
* subtle rarity-colored border

Examples:
common:
faint grey border

rare:
silver tint

ultra rare:
gold tint

hyper rare:
rainbow tint

This creates:
anticipation and chase desire.

========================================================
3.3 — TAP UNOWNED CARDS
=======================

Unowned cards MUST still be tappable.

Opening them launches:
“Card Preview Modal”

========================================================
SECTION 4 — CARD PREVIEW MODAL
==============================

For unowned cards.

========================================================
4.1 — PREVIEW CONTENT
=====================

Display:

* obscured or dimmed artwork
* rarity
* estimated value
* estimated pull rate
* wishlist toggle
* evolution chain if available
* card number
* set name

DO NOT show:
full attack/body text if hidden state is intended.

Goal:
tease the collectible.

========================================================
4.2 — CHASE PSYCHOLOGY
======================

This screen should emotionally create:
“I NEED THIS CARD.”

Use:

* subtle rarity glow
* premium spacing
* elegant typography

NOT:
aggressive animations.

========================================================
SECTION 5 — DEDICATED WISHLIST PAGE
===================================

Current issue:
Wishlisted cards disappear into nowhere.

========================================================
5.1 — NEW NAVIGATION ENTRY
==========================

Add:
“Wishlist”

Accessible from:

* Collection screen top navigation

Position:
near Statistics button.

========================================================
5.2 — WISHLIST PAGE DESIGN
==========================

Layout:
clean collectible gallery.

Group by:

* set
  OR
* rarity

Each card shows:

* artwork or silhouette
* rarity
* estimated value
* owned/unowned state

Owned wishlist cards:
small “Owned” badge.

========================================================
5.3 — EMPTY WISHLIST STATE
==========================

If no wishlisted cards:
show elegant empty state.

Example:
“No chase cards yet.”

Avoid:
dead empty page.

========================================================
SECTION 6 — WISHLIST REVEAL INTEGRATION
=======================================

If a revealed card is wishlisted:

Before reveal:

* subtle gold pulse
* slightly altered audio shimmer
* 150ms anticipation pause

DO NOT overdo.

This must feel:
special and exciting.

NOT:
arcade jackpot machine.

==================================

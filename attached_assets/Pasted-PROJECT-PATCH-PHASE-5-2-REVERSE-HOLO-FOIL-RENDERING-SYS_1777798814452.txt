PROJECT PATCH — PHASE 5.2
REVERSE HOLO FOIL RENDERING SYSTEM

Goal:

Reverse holo cards already exist in the simulation engine via:

isReverseHolo: true

This patch adds a **visual foil shimmer effect** so reverse holo cards look special when revealed.

This effect must be purely **visual** and must NOT modify any simulation logic.

Reverse holo should feel like a **mini-hit moment** during pack opening.

---

CRITICAL RULES

Do NOT modify:

RNG engine
packSimulation.js slot logic
pity system
rarity probabilities
API loader
cardPoolManager

Only modify:

renderers/cardRenderer.js
renderers/cardRevealAnimator.js
style.css

---

SECTION 1 — ADD REVERSE HOLO CSS EFFECT

Open:

style.css

Add the following styles:

.reverse-holo-overlay {

position: absolute
top: 0
left: 0
width: 100%
height: 100%
pointer-events: none
mix-blend-mode: overlay
opacity: 0.35

background-image: repeating-linear-gradient(
45deg,
rgba(255,255,255,0.05) 0px,
rgba(255,255,255,0.05) 2px,
transparent 2px,
transparent 6px
);

animation: reverseHoloShift 3s linear infinite;

}

---

SECTION 2 — ADD FOIL SHIMMER ANIMATION

Still inside style.css add:

@keyframes reverseHoloShift {

0% {
background-position: 0px 0px;
}

100% {
background-position: 200px 200px;
}

}

---

SECTION 3 — ADD LIGHT SWEEP EFFECT

Add an additional overlay layer:

.reverse-holo-sweep {

position: absolute
top: -40%
left: -60%
width: 200%
height: 200%

background: linear-gradient(
120deg,
transparent 30%,
rgba(255,255,255,0.35) 50%,
transparent 70%
);

animation: holoSweep 4s infinite;

pointer-events: none

}

@keyframes holoSweep {

0% { transform: translateX(-100%) rotate(25deg); }

100% { transform: translateX(100%) rotate(25deg); }

}

---

SECTION 4 — APPLY EFFECT IN CARD RENDERER

Open:

renderers/cardRenderer.js

Locate the function that creates the card element.

After the card image element is created, add:

if (card.isReverseHolo) {

const foil = document.createElement("div")
foil.className = "reverse-holo-overlay"

const sweep = document.createElement("div")
sweep.className = "reverse-holo-sweep"

cardElement.appendChild(foil)
cardElement.appendChild(sweep)

}

This ensures reverse holo cards get the foil overlay.

---

SECTION 5 — ADD SUBTLE SOUND / VISUAL EMPHASIS

Open:

renderers/cardRevealAnimator.js

Inside the reveal logic where a card becomes visible, add:

if (card.isReverseHolo) {

cardElement.classList.add("reverse-holo-highlight")

}

---

SECTION 6 — ADD REVERSE HOLO GLOW CLASS

In style.css add:

.reverse-holo-highlight {

box-shadow:
0 0 12px rgba(255,255,255,0.25),
0 0 25px rgba(200,200,255,0.15);

}

This glow should be subtle — reverse holo should feel special but **not stronger than hit cards**.

---

SECTION 7 — ENSURE LAYER STACKING WORKS

Add to card container CSS if not already present:

.card-container {

position: relative
overflow: hidden

}

This ensures foil overlays align with the card.

---

SECTION 8 — TESTING

Open a pack.

Expected result:

Slots 1–5 → normal cards
Slots 6–7 → normal cards
Slot 8 → **reverse holo foil shimmer effect**
Slot 9 → rare
Slot 10 → hit slot

The reverse holo card should display:

• reflective foil pattern
• light sweep animation
• subtle glow highlight

---

SECTION 9 — DESIGN GOAL

Reverse holo should feel like:

"oh nice, shiny!"

But NOT overshadow:

Ultra Rare
Special Illustration Rare
Hyper Rare

Hit slot effects will be implemented in Phase 5.3.

---

EXPECTED RESULT

Pack opening now contains **two excitement beats**:

1️⃣ Reverse holo shimmer reveal
2️⃣ Final hit card reveal

This improves the emotional pacing of pack openings and makes the simulator feel closer to real Pokémon TCG pack openings.

END PATCH

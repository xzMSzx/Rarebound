PROJECT UPDATE — PHASE 4 REAL POKÉMON CARD INTEGRATION

This update upgrades the pack simulator to use real Pokémon Trading Card Game cards.

Cards will be fetched from the Pokémon TCG API and cached locally when a set is selected.  
The existing simulation engine, pack RNG, and overlay reveal system must remain unchanged.

The goal is to replace placeholder rarity cards with real Pokémon card images.

---

CRITICAL RULE

Do NOT modify:

engine RNG logic  
pity system  
pack slot probabilities  
overlay animation system  
skip summary logic  

Only add a card data layer and integrate it with the reveal UI.

---

API SOURCE

Use the public Pokémon TCG API:

https://api.pokemontcg.io/v2/cards

No authentication key is required for small requests.

Each card entry contains fields such as:

name  
rarity  
set  
images.large

Use images.large for card artwork.

---

NEW DIRECTORY

Create a new folder:

/data

Files inside:

setLoader.js  
cardPoolManager.js  
rarityMapper.js

---

FILE — setLoader.js

Responsible for fetching cards from the Pokémon API.

Function:

async function fetchSetCards(setId)

This should call:

https://api.pokemontcg.io/v2/cards?q=set.id:{setId}

Return the card array from the API response.

Example returned card object:

{
  name: "Pikachu",
  rarity: "Rare",
  images: {
    large: "card image url"
  }
}

---

FILE — rarityMapper.js

Map Pokémon API rarity names into the simulator rarity tiers.

Example mapping:

Common → common  
Uncommon → common  
Rare → rare  
Rare Holo → epic  
Rare Ultra → legendary  
Rare Secret → legendary

Function:

function mapPokemonRarity(apiRarity)

Return one of:

common  
rare  
epic  
legendary

If rarity is missing, default to common.

---

FILE — cardPoolManager.js

This module stores the currently loaded set.

Variables:

currentSetId  
cachedCards

Function:

async function loadSet(setId)

If the set is already cached, return cached data.

Otherwise:

1. fetch cards via setLoader  
2. map rarities using rarityMapper  
3. store cards grouped by rarity

Example structure:

{
  common: [],
  rare: [],
  epic: [],
  legendary: []
}

Store this in cachedCards.

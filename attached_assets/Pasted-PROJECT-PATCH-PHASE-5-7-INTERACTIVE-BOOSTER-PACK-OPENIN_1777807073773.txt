PROJECT PATCH — PHASE 5.7
INTERACTIVE BOOSTER PACK OPENING SYSTEM (TCGP STYLE)

Goal:

Replace the current **instant pack open** with a **fully interactive booster pack opening sequence**.

The pack should:

• Display the correct artwork for the selected set
• Shake subtly before ripping
• Allow the user to drag across the **top seam only**
• Trigger a tear animation with sound
• Split the foil wrapper
• Slide the cards upward from inside the pack
• Prompt the user to swipe upward
• Then start the existing **card reveal system**

This must integrate with all previous phases without modifying core gameplay logic.

---

IMPORTANT USER EXPERIENCE FLOW

The opening sequence must follow this exact order:

pack appears
↓
foil crinkle sound
↓
subtle pack shake
↓
user drags across seam
↓
tear animation
↓
paper rip sound
↓
top foil splits
↓
cards slide upward
↓
swipe prompt appears
↓
user swipes up
↓
existing reveal system begins

This flow should be implemented exactly.

SECTION 1 — CREATE PACK ART DIRECTORY

/assets/packs

This folder stores booster pack artwork.

Expected files:

sv4pt5.png
swsh7.png
swsh11.png
sv3pt5.png
sv2.png

Each file corresponds to the setId already used by the simulator.

---

SECTION 2 — AUTOMATIC PACK ART FETCH

Create script:

/scripts/setupPackArt.js

Purpose:

Automatically download booster pack artwork from public Pokémon TCG resources.

The script should:

1. Define mapping of setId → artwork URL
2. Download the artwork
3. Resize if necessary
4. Save to /assets/packs

Example mapping:

```javascript
const PACK_ART = {

sv4pt5: "https://images.pokemontcg.io/sv4pt5/logo.png",
swsh7: "https://images.pokemontcg.io/swsh7/logo.png",
swsh11: "https://images.pokemontcg.io/swsh11/logo.png",
sv3pt5: "https://images.pokemontcg.io/sv3pt5/logo.png",
sv2: "https://images.pokemontcg.io/sv2/logo.png"

}
```

Download each image and save as:

/assets/packs/{setId}.png

Run with:

node scripts/setupPackArt.js

If artwork already exists, skip download.

SECTION 8 — DRAG TO RIP

Inside packOpeningController.js:

```javascript
let startX = null

seam.addEventListener("pointerdown", e => {

startX = e.clientX

})

seam.addEventListener("pointerup", e => {

const distance = e.clientX - startX

if(Math.abs(distance) > 80){

triggerRip()

}

})
```

This simulates dragging across the seam.

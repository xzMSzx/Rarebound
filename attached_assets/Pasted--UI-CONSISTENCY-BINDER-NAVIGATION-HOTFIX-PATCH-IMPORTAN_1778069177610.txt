## SECTION 2 — FIX PACK TITLE LAYOUT

Inside vendor pack cards:

Create a fixed-height title region.

Example:

min-height: 48px;
display: flex;
align-items: center;
justify-content: center;
text-align: center;
line-height: 1.2;
}

This ensures:

* 1-line titles remain vertically centered
* 2-line titles occupy same total space
* ALL cards remain equal height

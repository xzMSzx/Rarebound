## 2026-05-20 - Custom Accordion Accessibility
**Learning:** In hybrid architectures combining React and vanilla JS UI modules (like in `msge-lite/ui`), custom accordions implemented via standard HTML and JS event listeners often rely purely on CSS classes (e.g., `is-open`) for visual toggling. This leaves screen readers unaware of the interactive state and the relationship to the content block.
**Action:** Always verify that custom expandable UI elements programmatically update their `aria-expanded` attribute based on state and use `aria-controls` to properly link the trigger button to the dynamically shown/hidden body content's ID.
